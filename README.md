# Chip-8

This repo explores the creation of a chip-8 virtual cpu as a purely academic project with an active goal of creating a project that can be done without the use of AI. Due to the reach of AI in the current day and age I have found that my own ability to understand why I make the design decisions that I make decays, and so I am making this project specifically to reverse this decay by experiencing all of the good and bad design choices in a low stakes sandbox. In this I will be creating this virtual cpu based on a decades old specification.
